## Sinatra + HAML + Bulma (MVC)
Pequeña base mvc con el framework de ruby Sinatra

### Puesta en marcha

Instalar gemas (Dependencias)
``` 
bundle install 
ó 
bundle
```

Crear base de datos

```
rake db:migrate
rake db:seed
```

> El seed se tiene que configurar segun lo que desees realizar

Ejecutar proyecto

``` 
shotgun config.ru
```
