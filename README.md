## Sinatra + HAML + Bulma (MVC)
Pequeña base mvc con el framework de ruby Sinatra

### Puesta en marcha

Instalar gemas (Dependencias)
``` 
bundle install 
ó 
bundle
```

Crear base de datos

```
rake db:migrate
```

Ejecutar proyecto

``` 
shotgun config.ru
```
